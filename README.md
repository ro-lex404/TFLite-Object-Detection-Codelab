Currently it consists of a currency detection app to identify currency denominations of indian notes 10rs, 20rs, 50rs, 100rs, 200rs and 500rs to assist visually impaired.

Model used yolov8n. Trained on around 16k images.

99.2 precision

Still struggles in misclassification sometimes.
